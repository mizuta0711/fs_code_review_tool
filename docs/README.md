# FSCodeReviewTool

AIによるシンプルなコードレビューアプリケーション

## 概要

FSCodeReviewToolは、プログラミング初級者〜中級者が作成したコードに対して、AIが優しい表現でコードレビューを行うWebアプリケーションです。

### 主な特徴

- **ログイン不要** - 誰でもすぐに使える
- **シンプルな操作** - コードを貼り付けて「レビュー実行」を押すだけ
- **複数AI対応** - Gemini / Azure OpenAI / Claude から選択可能
- **複数ファイル対応** - 複数ファイルのプロジェクトも一括レビュー
- **ドラッグ&ドロップ** - ファイルをドロップするだけで読み込み
- **プロンプト管理** - レビュー観点をカスタマイズ可能

---

## 使い方

### 1. コードレビューを実行する

1. トップページ（`/`）にアクセス
2. コード入力エリアにレビューしたいコードを貼り付け
   - または、ファイルをドラッグ&ドロップで追加
3. 「プロンプト選択」でレビュー観点を選択
4. 「AIプロバイダー」でレビューに使用するAIを選択
5. 「レビュー実行」ボタンをクリック
6. レビュー結果がコメント付きコードとして表示される

### 2. 複数ファイルをレビューする

1. ファイルタブの「+」ボタンで新規ファイルを追加
2. または、複数ファイルを一度にドラッグ&ドロップ
3. 各ファイルにコードを入力
4. 「レビュー実行」で全ファイルを一括レビュー

### 3. レビュー結果を取得する

- **コピー** - 現在表示中のファイルをクリップボードにコピー
- **ダウンロード** - 現在表示中のファイルをダウンロード
- **全ファイルDL** - 全ファイルをZIPでまとめてダウンロード

## 設定

### AIプロバイダー設定（`/settings`）

複数のAIプロバイダーを登録・管理できます。

#### 対応プロバイダー

| プロバイダー | 必須設定 |
|-------------|---------|
| Google Gemini | APIキー |
| Azure OpenAI | APIキー、エンドポイント、デプロイメント名 |
| Anthropic Claude | APIキー |

#### 設定手順

1. 「新規追加」ボタンをクリック
2. 表示名、プロバイダー種別を選択
3. APIキーと必要な設定を入力
4. 「保存」をクリック

#### デフォルト設定

- 「デフォルトに設定」で、レビュー時に初期選択されるプロバイダーを指定
- レビュー実行時に、使用するプロバイダーを都度選択することも可能

#### 利用パスワード（オプション）

- 各プロバイダーに利用パスワードを設定可能
- パスワード設定済みのプロバイダーは、レビュー実行時にパスワード入力が必要
- パスワード未設定のプロバイダーは、そのまま利用可能

---

### プロンプト管理（`/prompts`）

レビュー時に使用するプロンプト（指示文）を管理できます。

#### 機能

- **プロンプト一覧** - 登録済みプロンプトの確認
- **新規作成** - 新しいレビュー観点のプロンプトを作成
- **編集** - 既存プロンプトの内容を変更
- **デフォルト設定** - レビュー時に初期選択されるプロンプトを指定

#### プロンプトの書き方

プロンプトには以下の変数が使用できます：

- `{code}` - レビュー対象のソースコード
- `{filename}` - ファイル名

## 技術スタック

- **フロントエンド**: Next.js 15, React 19, TypeScript, Tailwind CSS
- **UIライブラリ**: shadcn/ui
- **エディタ**: Monaco Editor
- **データベース**: PostgreSQL (Neon)
- **ORM**: Prisma

## ローカル開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# 本番サーバー起動
npm run start
```

### 環境変数

`.env`ファイルに以下を設定：

```
DATABASE_URL=postgresql://...
ENCRYPTION_KEY=32文字のランダム文字列
```
