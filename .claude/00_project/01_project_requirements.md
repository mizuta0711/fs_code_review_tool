# プロジェクト要件定義

## プロジェクト概要

### FSCodeReviewTool とは
FSCodeReviewTool は、AIを活用したコードレビュー支援Webアプリケーションです。
開発者がソースコードを投稿し、AIによる自動レビューと改善提案を受けることができます。

### ターゲットユーザー
- 個人開発者
- 小規模チームの開発者
- コードレビューを効率化したいエンジニア
- コード品質を向上させたい学習者

## 機能要件

### コア機能

#### 1. コードレビュー機能
- **コード投稿**: テキストエリアまたはファイルアップロード
- **言語自動検出**: プログラミング言語の自動判別
- **AIレビュー**: Google Gemini / OpenAI APIによるコード分析
- **改善提案**: 具体的な改善点とサンプルコードの提示
- **レビュー評価**: コード品質スコアの算出

#### 2. プロジェクト管理機能
- **プロジェクト作成**: レビュー対象プロジェクトの登録
- **ファイル管理**: プロジェクト内ファイルの管理
- **GitHub連携**: リポジトリからのコード取得（将来対応）

#### 3. レビュー履歴機能
- **履歴保存**: 過去のレビュー結果の保存
- **履歴検索**: レビュー履歴の検索・フィルタリング
- **比較機能**: 過去のレビューとの比較

#### 4. ユーザー管理機能
- **ユーザー登録**: OAuth認証（GitHub/Google）
- **プロフィール**: ユーザー情報管理
- **設定**: レビュー設定のカスタマイズ

### 将来対応機能
- チーム機能（複数ユーザーでのコラボレーション）
- PRレビュー連携（GitHub PR連携）
- カスタムルール設定（レビュールールのカスタマイズ）
- 多言語対応（i18n）

## 非機能要件

### パフォーマンス
- ページ読み込み: 3秒以内
- レビュー処理: 30秒以内
- 同時接続: 100ユーザー以上

### セキュリティ
- 認証: OAuth 2.0
- データ暗号化: HTTPS必須
- 入力検証: 全入力値のバリデーション

### 可用性
- 稼働率: 99.5%以上
- バックアップ: 日次バックアップ

## 技術要件

### フロントエンド
- Next.js 15 (App Router)
- React 19
- TypeScript 5
- Tailwind CSS v4
- shadcn/ui

### バックエンド
- Next.js API Routes
- Prisma ORM
- PostgreSQL

### 外部サービス
- 認証: NextAuth.js (GitHub/Google OAuth)
- AI: Google Gemini API / OpenAI API
- ホスティング: Vercel
- データベース: Railway / Supabase
